# 💰 MBG Converter  
**Makan Bergizi (tidak) Gratis Converter**

A simple web app to convert **Rupiah (IDR)** into **MBG units** and vice versa.

> ⚠️ This project is a satirical / educational tool inspired by the controversial Indonesian government program.

## 📌 Concept

- **1 MBG = Rp 1,200,000,000,000 (1.2 Trillion Rupiah)**
- MBG is represented in **time units (days)** for easier visualization of large budgets.

---

## ✨ Features

- 🔄 Convert **Rupiah → MBG (days)**
- 🔄 Convert **MBG → Rupiah**
- ⏱ Detailed time breakdown:
  - Days
  - Hours
  - Minutes
  - Seconds
- 🇮🇩 Auto-format Rupiah input (1.000.000 style)

---

## 🖥️ Demo Usage

### Convert Rupiah → MBG

Input:
```
Rp 1,200,000,000,000
```

Output:
```
1.000000 Hari
= 1 hari 0 jam 0 menit 0 detik
```

---

### Convert MBG → Rupiah

Input:
```
2
```

Output:
```
Rp 2,400,000,000,000
```

## 🚀 How to Run

### Option 1 — Open directly
Just open the HTML file in your browser:
```
index.html
```

### Option 2 — Simple local server
```bash
python3 -m http.server 8000
```

Then open:
```
http://localhost:8000
```

## 💡 Idea Behind

Large government budgets are hard to grasp.

By converting them into **"days of MBG"**, this tool helps:
- Make numbers more tangible
- Encourage public discussion
- Provide a simple mental model
